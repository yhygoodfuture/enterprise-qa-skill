# 财务报销制度

## 报销范围

| 类别 | 说明 | 示例 |
|------|------|------|
| 差旅费 | 因公出差产生的费用 | 机票、酒店、交通 |
| 业务招待费 | 客户相关业务活动 | 餐叙、礼品 |
| 办公用品 | 工作所需物品 | 电脑、外设、书籍 |
| 培训费 | 学习提升费用 | 课程、会议门票 |
| 通讯费 | 工作通讯费用 | 电话费、流量包 |

## 报销标准

### 差旅费标准

| 项目 | 标准 | 说明 |
|------|------|------|
| 机票 | 经济舱 | 提前 7 天预订，特殊情况需审批 |
| 高铁 | 二等座 | - |
| 酒店 | 一线城市≤500 元/天 | 北京、上海、广州、深圳 |
| 酒店 | 其他城市≤300 元/天 | - |
| 餐补 | 出差 100 元/天 | 按自然日计算 |
| 市内交通 | 实报实销 | 优先公共交通 |

### 业务招待费标准

| 项目 | 标准 | 说明 |
|------|------|------|
| 餐叙 | ≤500 元/人 | 需事前审批 |
| 礼品 | ≤300 元/份 | 需事前审批 |
| 单次上限 | ≤2000 元 | 超额需 VP 审批 |

### 办公用品标准

| 项目 | 标准 | 说明 |
|------|------|------|
| 笔记本电脑 | ≤15000 元 | 按需申请 |
| 显示器 | ≤3000 元 | 最多 2 个 |
| 书籍 | ≤500 元/月 | 需与工作相关 |

## 报销流程

```
系统提交 → 主管审批 → 财务审核 → 打款（5 工作日）
```

### 各阶段说明

| 阶段 | 处理人 | 时限 |
|------|-------|------|
| 系统提交 | 申请人 | - |
| 主管审批 | 直属主管 | 2 工作日 |
| 财务审核 | 财务部 | 2 工作日 |
| 打款 | 出纳 | 5 工作日 |

### 报销所需材料

| 类型 | 材料 |
|------|------|
| 差旅费 | 发票、行程单、出差申请 |
| 业务招待 | 发票、水单、事前审批 |
| 办公用品 | 发票、入库单 |
| 培训费 | 发票、培训通知 |

## 注意事项

1. **发票要求**：必须是增值税专用/普通发票，抬头正确
2. **时效要求**：费用发生后 30 日内提交，逾期不予受理
3. **虚假报销**：一经查实，严肃处理，直至辞退

## 常见问题

**Q: 加班打车可以报销吗？**
A: 21:00 后加班打车可以报销，需在系统中关联加班申请。

**Q: 团建费用怎么报销？**
A: 团建费用需提前申请，按批准的预算执行，超出部分自理。

**Q: 发票丢了怎么办？**
A: 需提供发票底联复印件 + 情况说明，主管 + 财务双审批。
