## 技术分享一：ReMe 架构升级

**演讲人**: 张三

### 背景

ReMe 1.0 上线后遇到以下问题：
- 单租户架构，无法支持多客户
- Session 无法持久化，重启丢失
- 向量检索性能瓶颈

### 升级内容

| 特性 | 1.0 | 2.0 |
|------|-----|-----|
| 租户支持 | 单租户 | 多租户 |
| Session | 内存 | ChromaDB 持久化 |
| 检索性能 | 100ms | 10ms |

### 技术要点

- 租户隔离：Schema 级别隔离
- Session 持久化：ChromaDB 文件存储
- 向量索引：HNSW 算法

## 技术分享二：MCP 协议实践

**演讲人**: 李四

### 什么是 MCP？

Model Context Protocol (MCP) 是 AI 助手与工具之间的统一接口协议。

### 我们的实践

```
Claude Code ←MCP→ MCP Server ←→ Tools
                                    ├── Database
                                    ├── Vector Store
                                    └── File System
```

### 收益

- 工具接入标准化
- 支持动态发现工具
- 便于扩展
