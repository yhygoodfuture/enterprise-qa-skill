# 晋升评定标准

## 职级体系

| 职级 | 称号 | 典型年限 |
|------|------|---------|
| P4 | 初级工程师 | 应届~2 年 |
| P5 | 工程师 | 2~5 年 |
| P6 | 高级工程师 | 5~8 年 |
| P7 | 专家工程师 | 8~12 年 |
| P8 | 高级专家 | 12~15 年 |
| P9 | 资深专家 | 15 年 + |
| P10 | 研究员 | 特殊贡献 |

## 晋升条件

### P4 → P5

| 条件 | 要求 | 说明 |
|------|------|------|
| 工作年限 | 入职满 6 个月 | 或同等级满 1 年 |
| 绩效要求 | 连续 2 季度≥B | 无 C 或 D 评价 |
| 学习任务 | 完成导师指定任务 | 通过考核 |

### P5 → P6

| 条件 | 要求 | 说明 |
|------|------|------|
| 工作年限 | 入职满 1 年 | 或 P5 满 2 年 |
| 绩效要求 | 连续 2 季度 KPI≥85 | 或年度平均≥85 |
| 项目经验 | 主导或核心参与≥3 个 | 至少 1 个完整周期 |
| 事故记录 | 无重大事故 | P0/P1 级事故 |

### P6 → P7

| 条件 | 要求 | 说明 |
|------|------|------|
| 工作年限 | P6 满 2 年 | 破格需特批 |
| 绩效要求 | 连续 4 季度 KPI≥90 | 或年度 2 个 S |
| 项目经验 | 主导项目≥2 个 | 作为技术负责人 |
| 技术贡献 | 技术突破/专利/论文 | 至少 1 项 |

### P7 → P8

| 条件 | 要求 | 说明 |
|------|------|------|
| 工作年限 | P7 满 3 年 | - |
| 绩效要求 | 年度绩效至少 2 个 S | 连续 2 年 A 以上 |
| 业务贡献 | 显著业务贡献 | 营收/效率提升≥30% |
| 影响力 | 团队培养/技术分享 | 培养 2 名以上骨干 |

## 破格晋升

满足以下条件之一可申请破格晋升：

1. **技术突破**：解决重大技术难题，获得专利或顶会论文
2. **业务贡献**：主导项目带来显著营收增长（≥1000 万）
3. **紧急需求**：业务扩张需要，岗位紧急空缺

破格晋升需经技术委员会评审，CEO 批准。

## 晋升流程

1. **提名**：主管提名或个人申请（每年 3 月、9 月）
2. **评审**：技术委员会评审
3. **答辩**：候选人现场答辩
4. **公示**：结果公示 5 工作日
5. **生效**：次月 1 日生效
